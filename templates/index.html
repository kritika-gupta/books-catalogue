Use our API!
We provide an API to access the average rating a book receives on our website.

To access the title, author, year, average rating, number of ratings for a book, send a GET request to the following link : '/api/<isbn>' where the parameter 'isbn' is the ISBN of the book you are requesting data for.
If the ISBN is present in our database you will get a JSON response with keys : 'isbn', 'title', 'author', 'year', 'average_score', 'review_count'.
